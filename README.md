## 🚧 Proyecto actual

- `Traductor de pantalla en vivo`
  - Captura texto de la pantalla en Windows
  - OCR con `rapidocr-onnxruntime`
  - Traduce con `deep-translator`
  - Muestra la traducción en un overlay transparente sin bloquear otras apps
  - Usa `pystray`, `pywin32`, `mss`, `Pillow` y procesamiento en background

## 🧩 Proyecto destacado

- `Traductor en vivo de pantalla`
  - Aplicación modular en Python
  - Estructura: `main.py`, `capture.py`, `ocr.py`, `translate.py`, `overlay.py`, `tray.py`, `worker.py`
  - Funcionalidad: captura → OCR → traducción → overlay
  - Configuración persistente en `config.py`
  - Soporte para pausar, limpiar caché y ajustes rápidos
